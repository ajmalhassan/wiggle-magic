Welcome to Wiggle Magic

Turn any web page into an AI selection canvas with a flick of the cursor.

  1. 1

    Wiggle

    Briskly shake your cursor anywhere on a page to enter selection mode.

  2. 2

    Pick

    Click any element — text, image, code, table cell — to add it. Click again to remove.

  3. 3

    Magic

    Click the floating Magic pill (or press Enter) to ask Gemini Nano about your selection.

Don't see the download option, or it's stuck on "unavailable"?

In most cases Gemini Nano just needs the on-device model pulled — the button above does that. If it's not showing or fails, walk through these manual steps:

  1. Confirm you're on Chrome 138 or newer (check chrome://settings/help).
  2. Open chrome://flags/#prompt-api-for-gemini-nano → set to Enabled.
  3. Open chrome://flags/#optimization-guide-on-device-model → set to Enabled BypassPerfRequirement.
  4. Relaunch Chrome.
  5. Open chrome://components/, find Optimization Guide On Device Model, click Check for update. Wait for the ~2 GB model to download.
  6. Reload this Settings page — status should flip to available.

Full reference: Chrome's built-in AI / Prompt API docs.
