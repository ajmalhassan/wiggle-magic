The gesture

Briskly shake the cursor side-to-side anywhere on a page. The cursor swaps to a glowing pointer, the viewport edges light up — you're in selection mode. Click to pick elements; click again to deselect. Move onto the floating Magic pill (or press Enter) to commit.

Tip The gesture needs at least 4 direction reversals within ~600 ms, in a small bounding box, at average speed ≥ 0.25 px/ms. Loose enough to be deliberate, strict enough not to fire on normal mouse paths.

Where your data goes

Gemini Nano

Runs entirely on your machine. Selections and questions never leave the device. Model weights are downloaded once by Chrome itself and cached system-wide.

Bring your own key

Prompts go directly from your browser to the provider you chose (OpenAI / Anthropic / Gemini). There is no Wiggle Magic server. Your key lives in chrome.storage.sync on this device.

Saved answers live in chrome.storage.local on this device. Export them as Markdown anytime from the toolbar popup.

Troubleshooting

"Gemini Nano is unavailable on this device"

Open Settings — if you see a Download model (~2 GB) button, click it and wait. If not:

  • Confirm Chrome ≥ 138 (chrome://settings/help)
  • Enable chrome://flags/#prompt-api-for-gemini-nano
  • Enable chrome://flags/#optimization-guide-on-device-model as Enabled BypassPerfRequirement
  • Relaunch Chrome
  • Open chrome://components/, find Optimization Guide On Device Model, click Check for update

Full reference: Chrome built-in AI / Prompt API docs.

Wiggle doesn't fire on a specific site
  • The extension doesn't run on chrome:// pages, the Chrome Web Store, the New Tab page, or the PDF viewer.
  • It runs in the top frame only, so iframes (Stripe checkouts, embedded videos) won't trigger.
  • Some sites (Figma, Notion, Google Docs) own the cursor — the gesture should still fire but the cursor swap may look off.
BYOK call failed
  • Make sure the key matches the provider you selected (OpenAI key won't work against Anthropic).
  • Anthropic requires the anthropic-dangerous-direct-browser-access header — the extension sets it, but the org policy on your key may still block browser calls. Try a key from a different account.
  • Rate-limited keys silently return an empty stream. Check your provider's usage dashboard.
Saved answers aren't appearing
  • Save only triggers from the Save button on a successful answer — not automatically.
  • The cap is 500 entries; older ones drop off the bottom. Export to Markdown to keep them long-term.
  • Browsers in incognito don't have access to your normal chrome.storage.local — that's by design.
The cursor looks wrong on this site

Some sites set a custom CSS cursor at high specificity. We override it with cursor: none !important on body.wm-active *, but if the site uses an inline style on a specific element, it can still bleed through. Wiggle still works either way — the AI flow is independent of the cursor look. File an issue with the site URL and we'll add a workaround.

Tunable thresholds

The gesture detector lives in content.js with a small opts object you can tweak if you fork:

  • windowMs: 600 — sliding-window length
  • minReversals: 4 — direction flips required
  • maxRadius: 220 — gesture confinement (px)
  • minSpeedPxMs: 0.25 — avg pointer speed
  • minSamples: 5 — minimum samples before evaluating
  • cooldownMs: 1200 — quiet period after a trigger
